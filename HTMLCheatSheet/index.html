HTML Elements

Tag Description
<!DOCTYPE html> Defines standard of HTML document.
<html> Creates HTML document.
<head> Sets info about HTML document that is not visible to the user.
<body> Defines the visible content to the user.
<h?> Creates heading text. Largest: h1 | Smallest: h6
<p> Creates a new paragraph
<strong> Creates bold text.
<em> Creates italicized text.
<ul> Creates an unordered list.
<ol> Creates ordered list.
<li> Encompasses list item. NOTE: Must be used within <ol> or <ul> element.
<a href="URL"> Creates a hyperlink to another page or id within HTML document.
<img src="URL" alt="Description"> Adds an image to HTML document and contains attribute to describe the image.

CSS Selectors

Tag Description
p {} Type Selector - matches the type of the element in the HTML document.
* {} Universal Selector - selects all elements of any type.
.class {} Class Selector - select elements with the matching class attribute
#id {} ID Selector - selects elements with the matching id attribute
